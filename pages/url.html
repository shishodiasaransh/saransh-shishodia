URL Shortner Website

This website have login and signup pages which led to the URL shortner page. This is a free service and anyone can use this and authentication is required. The data is stored in online atlas mongodb cluster. It also shows the previous urls visited and shortened also the total clicks.

Services provided:

  • Register and Login as Maid
  • Authentication
  • Shortened URL
  • Data about previous urls
