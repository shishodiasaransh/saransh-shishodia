Bus Reservation System

This is a Bus Reservation System simply build with the help of SQL.Reservation as well as printing of passenger receipt. This system allows the user to look for vacant seat quickly.

Services provided:

  • Checks if seat availability
  • on availability it gives yes as output
  • reserves the seat to the customer
  • if not available gives no as output

    • Technologies used:

  • SQL (DataBase)
  • PL/SQL block
